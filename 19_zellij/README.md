# tmux key

```bash
# 새창 추가
Ctrl+ b + c

# vs 창추가 (right)
Ctrl + b + %

# sp 창추가 (down)
Ctrl + b + "

# 창 전환(Previous)
Ctrl + b + p

# 창 전환(Next)
Ctrl + b + n
```
