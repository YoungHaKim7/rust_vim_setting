# Helix 주로 쓰는 command

```
:set cursorline true   " 한줄 내가 코드 쓰는거 줄 강조 됨
:set color-modes true      "insert, normal, sellect mode  색깔로 이쁘게
:config-open  " 설정파일 열기
:config-reload " 설정파일 reload 바로 적용하기
```
